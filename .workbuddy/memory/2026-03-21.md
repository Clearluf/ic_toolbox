## 数据切片器开发（Tool #2）

- 实现 `DataSlicerFrame` 类，接入侧边栏第二个工具位
- 功能：输入 HEX / BIN 数据 + 总位宽，按指定 bit 宽度切片
- 切片宽度：4 / 8 / 16 / 32 / 64 / 自定义，快捷按钮选择
- 字节序：Big-Endian / Little-Endian 单选切换
- 输出列：序号、Bit Range、HEX、BIN（可关闭）、DEC-U、DEC-S（可关闭）
- 点击 HEX/BIN/DEC 列可直接复制
- 结果区带滚动条，支持鼠标滚轮
- 输入支持 `_` 分隔符、`0x`/`0b` 前缀、Verilog `'h`/`'b` 格式

- 新增**前导零开关**：勾选后 BIN/OCT/HEX 输出补零至完整位宽，取消勾选则不补零；Verilog 格式始终使用完整补零值
- 新增**移位操作面板**：
  - 支持逻辑左移 `<<`、逻辑右移 `>>`、算术右移 `>>>`（有符号模式下保留符号位）
  - 快捷位数按钮：1 / 2 / 4 / 8
  - 结果同步显示 DEC / HEX / BIN
  - 支持「↩ 回填」将移位结果填回转换器

## 数据比对器开发（Tool #3）

- 实现 `DataDiffFrame` 类，接入侧边栏第三个工具位
- 功能：两路 HEX/BIN 输入，逐 bit 比对
- 差异位以红色高亮显示（背景 #3d1a1a），相同位绿色

- 底部统计：差异段数、差异 bit 数、差异率
